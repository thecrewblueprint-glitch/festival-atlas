About this project

Production Atlas is a project of Deadhang Labor LLC, a live-event production labor company. It was built as a public resource to help production workers find festival opportunities, find employer contacts, and plan work calendars using publicly available information.

What Production Atlas is

Production Atlas is a public-facing work map for live-event production. It organizes festival dates, locations, approximate production windows, public producer/promoter information, employer contacts, and source references so contractors can decide which events are worth researching.

The app is designed for workers who need practical public information: where events happen, when they happen, who publicly produces or operates them, and which official company or labor pages are relevant to the search.

Built for public research

Production Atlas uses public-facing event, company, source, and directory information. It does not publish non-public work details, personal contact information, pay rates, lodging details, private referrals, or rumors.

Built for planning

The map, calendar, schedule, and opportunity pages help workers compare timing, region, and public employer contacts before deciding where to focus outreach.

Built for source review

Source links are centralized on the Sources page so opportunity and employer screens stay readable while public references remain available for review.

What it is not

Production Atlas is not a job board, hiring guarantee, private contact database, labor broker, legal advisor, travel agent, or substitute for direct verification with official event, employer, venue, or local-jurisdiction sources.
