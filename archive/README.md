# Archive

This folder stores retired or legacy files that are no longer part of the active Production Atlas runtime.

## Rule

Files outside this folder should be relevant to the currently working app.

Files moved here are kept for recovery/reference only. They should not be loaded by the live GitHub Pages app unless intentionally restored.

## Current architecture

The working multi-page app should use:

- `assets/atlas.css`
- `assets/atlas-core-v2.js`
- `assets/home-guide-page.js`
- page files such as `index.html`, `calendar.html`, `opportunities.html`, `branches.html`, `employers.html`, `iatse.html`, `matrix.html`, `analytics.html`, `sources.html`
- active data packages under `data/packages/`

## Archived categories

- `archive/legacy-runtime/` — old runtime patches from the single-page build phase.
