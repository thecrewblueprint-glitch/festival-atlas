How the Data Works

Production Atlas organizes public information into planning views for live-event production work research.

HomeGuideOpportunitiesCalendarMapEmployersSources

Data model

Each opportunity record can include a festival name, city, state, venue or site, public event dates, approximate production/build window, approximate strike/load-out window, public producer or promoter information, public source references, and display notes that are safe for public use.

Opportunities

Opportunity records identify events that may create live-event production work. These records are used by the Opportunities, Calendar, Map, and Schedule pages.

Departments

Department data remains in the repository and record model to explain production coverage, but broad department filtering is not the current public search focus.

Employers

Employer records point to official public company pages, careers pages, application pages, contact pages, or relevant organization pages. General employer listings are not treated as confirmed event-specific contracts.

Sources

Source references are kept on the Sources page. This keeps event popups and employer views focused on work mapping while still allowing public references to be reviewed separately.

Excluded public data

Production Atlas does not publish private contacts, personal emails, phone numbers, pay rates, hotel details, private field notes, client-sensitive information, NDA information, rumors, or private referrals.
