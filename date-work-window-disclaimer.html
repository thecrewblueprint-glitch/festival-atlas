Date & Work Window Disclaimer

Production Atlas separates public event dates from approximate planning windows.

Festival show dates

Festival show dates are based on public event information available in the app’s source set. Public dates can change, and workers should confirm current event dates with official sources before making travel, availability, or outreach decisions.

Approximate work windows

Approximate work windows are planning estimates intended to help workers think about possible load-in, show-call, and load-out timing. They are not confirmed crew-call dates, contract dates, or guaranteed work dates.

Calendar bars

On the calendar, the muted outer bar represents the approximate work window. The bright inner segment represents the public festival show dates.

Planning use

Use work windows to compare possible overlaps, routing conflicts, and regional timing. Confirm details before committing to travel or scheduling.

Source review

Use the Sources page to review public references connected to event records.
