## Scoring Method

This batch scores opportunity value based on work-year usefulness:

- lodging / housing signal
- paid travel / mileage / flight signal
- per diem signal
- work-window length
- hiring route clarity
- repeat or recurring EDM brand value
- source confidence

The score does not measure event popularity. It measures whether the target is likely to help fill open time in the work year.
